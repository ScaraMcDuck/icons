Many images from the Nuvola icon theme for KDE 3.x by David Vignoni (http://icon-king.com/?p=15)

Warning: Some of these icons may contain trademarks. Please check before you use them. The icons themselves are licensed under the LGPL.

These icons are in PNG versions, but SVG versions are preferred and available in some cases.

The PNG set is divided into 4 parts:

  • Application icons
  • Device icons
  • Filesystem icons
  • Filetype icons

Browse all icons, by subject depicted, here.

See also: Commons:Project Nuvola 2.0+

All images in this category should be recreated using vector graphics as an SVG file. This has several advantages; see Commons:Media for cleanup for more information. If an SVG form of this image is already available, please upload it. After uploading an SVG, replace this template with template {{Vector version available|new image name.svg}} in this image.
