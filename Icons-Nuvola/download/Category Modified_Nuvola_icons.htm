All SVG versions of Nuvola icons are modified, so they should be categorized under Category:Nuvola SVG icons, not here, as their status as "modified" is implied by not being a PNG.

All images in this category should be recreated using vector graphics as an SVG file. This has several advantages; see Commons:Media for cleanup for more information. If an SVG form of this image is already available, please upload it. After uploading an SVG, replace this template with template {{Vector version available|new image name.svg}} in this image.
